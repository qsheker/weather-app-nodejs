# Weather App

A weather application built with Node.js and Express.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with your OpenWeather API key:
```
OPENWEATHER_API_KEY=your_api_key_here
```

3. Run the app:
```bash
npm start
```

## Usage

Open `http://localhost:3000` and enter a city name to get weather information.

## Features

- Current weather data
- Country information
- Sunrise/sunset times
- Responsive design
